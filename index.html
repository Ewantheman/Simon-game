Simon Game

Press any keyboard key to start

1. Press any key on the keyboard to start the game.
2. Simon will give you the first color. Repeat it by clicking the same color box. This brings u to next level.
3. At next level, Simon will add one additional color. Repeat these two color by pressing the color boxes in order.
4. Continue playing as long as you can repeat each sequence of color correctly.
5. If you fail to repeat a sequence exactly, Simon responds with a"RAZZ" sound. This means you've lost, and the sequence of color ends. My high score is 13, can you beat it? :)
6. Click tutorial button again to close this tutorial
